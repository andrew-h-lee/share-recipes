
# Share-Recipes App Project
This RoR web app builds custom-made menu for users.

## Accomplished
* Adding Categories
* Create & Add Recipes
* Creating Data Relationship between Recipes & Categories
* Add Authentication
* Improving UI Views

## Prerequisites / Tools Used

* Rails (v4.1.6)

* Ruby (v2.1.0)

* MySQL

* Bootstrap (v3.3.0)

* Sass

* Devise
